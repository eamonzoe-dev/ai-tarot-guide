Buttons · 按钮

去掉高光，是质感差距最大的一处。

带渐变和立体反光的金色按钮，跟克制的画面不搭。改成扁平哑光——实心古铜，或奶白底＋细金边的描边按钮。

现在 · 弃用

渐变 + 内高光 + 投影，反光质感偏廉价，和整页的安静相冲。

改后 · 哑光

实心古铜，无渐变无投影，按下时轻微下沉。干净、克制、好用。

仪式 / 深底

Surfaces · 卡片

要么彻底扁平，要么一道极细边框。

别停在"只有淡淡阴影"的中间态——那是现在显得模糊的原因。三种面各有职责：抬起、凹陷、仪式。

抬起 · Raised 单牌解读 Vellum 底 + 1px 细描边。承载内容，安静浮起。
凹陷 · Well 解读便笺 Sand 底 + 强描边。用于输入框、引述、嵌套区。
仪式 · Ritual 三牌阵 墨绿实底。留给被选中、被强调、进入仪式的时刻。

Sigils · 符号

把卡背的圆＋十字＋菱形，长成一组线描。

同一种细线语言，用在分隔线、加载、空状态、栏目图标上——整站的个性就不只靠颜色和字撑着。

AI Reading · 本次解读

你今天的身体状态，更适合从"稳不稳、有没有被好好照顾"来判断。星币侍从提示你留意精神与基础体感。

它的核心不是夸张的好转或坏转，而是很朴素的星币能量：身体、作息、饮食、休息这些实际层面，和一种带着学习心态去观察自身的态度。

所以这张牌更像是在说：今天的状态有机会被你用很实际的方法照看住，但前提是别忽略身体发出的低声提示。

反思问题

今天最先显现出来的，是身体的疲惫，还是精神上的耗散？

Increment · 夜间模式

塔罗属于夜晚。
把它做成一个开关，而不是替换。

同一套点缀色翻到深底上：墨绿与古铜在夜里更显贵，仪式感更强。先把方向一的白天版做扎实，夜间版作为增量——等于多一个卖点。
